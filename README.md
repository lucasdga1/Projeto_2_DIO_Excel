# Projeto_2_DIO_Excel
Este projeto tem como objetivo criar uma ferramenta no Excel que ajude a organizar e reunir informações essenciais para a declaração de imposto de renda. A proposta é construir um agregador de dados no qual o usuário possa controlar suas entradas de maneira eficiente e validada, com menus de navegação, validações automáticas e funcionalidades extras, como links rápidos. A solução será completamente construída no Excel, com recursos que tornam a ferramenta robusta, mas com uma interface amigável e prática. Fruto do curso promovido pela DIO em parceria com o banco Santander

<a href="https://github.com/lucasdga1/Projeto_2_DIO_Excel/blob/main/Organizador_IRPF.xlsx">Organizador IRPF</a>

## Objetivos e aplicabilidade
Esta planilha auxilia o usuário a:
* Manter os dados pessoais atualizados
* Agregar e discriminar rendas por instituição financeira
* Detalhar rendas e suas fontes
* Controlar entradas de maneira eficiente e facilitada
